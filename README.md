# Web Application Frameworks

## 1. Personal Web Profile Page Using HTML
**Objective**: Design and develop a simple personal web profile page using only HTML. This assignment focuses on
understanding HTML document structure, semantic elements, hyperlinks, lists, tables, images, and forms
without using CSS or JavaScript.

## 2. Demonstrating Model-View-Controller (MVC) Architecture Using HTML
**Objective**: Create a simple website that conceptually represents the Model-View-Controller (MVC) architecture
using only HTML.

## 3. Responsive Online Book Store Homepage
**Objective**: Create a responsive online bookstore homepage that demonstrates the use of HTML elements, CSS styling,
Flexbox, Grid, and media queries.

## 4. Responsive College Event Management Web Page
**Objective**: Create a responsive webpage for a college technical fest that displays event information using semantic
HTML and responsive CSS layouts.
